# Бизнес-домен.

## Склад продуктов интернет-магазина
### 1. Источники заявок (ИБ, И31):
•	Заказы покупателей (онлайн-заявки). (Бесконечный источник, закон генерации - Пуассона)
________________________________________
### 2. Приборы (П32):
•	Приборы — это курьеры/комплектовщики заказов, которые отбирают продукты со склада для доставки клиентам.
•	Время обслуживания (подбора товаров для заказа) распределено равномерно.
________________________________________
### 3. Буфер (Д10З1):
•	Буфер = холодильник.
•	Организован по кольцу: новые продукты ставятся «следом за последним».
________________________________________
### 4. Дисциплина отказа (Д10О1):
•	Если полка заполнена и приходит новая партия, то система вытесняет старый продукт (тот, что «под указателем»).
•	Так как продукты имеют срок годности: старые быстрее портятся и выбрасываются, а новые занимают их место.
________________________________________
### 5. Дисциплина выбора прибора (Д2П1):
•	Приоритет комплектовщиков по номеру, например, старший сотрудник всегда получает заказы первым.
________________________________________
### 6. Дисциплина выбора заявок (Д2Б1 – FIFO):
•	Продукты со склада для заказов берутся в порядке поступления (FIFO), что соответствует реальной логистике: «первым пришёл — первым ушёл» (чтобы не залеживались).
________________________________________
### 7. Диспетчер:
•	Менеджер склада.
